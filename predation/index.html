Pinniped Predation Sampling Demo

Demonstrating event duration bias in spatio-temporal sampling of predation events

How to Use This Demo

  1. Set Parameters: Choose temporal units (rows), predation probability (p), and sampling fraction
  2. Create Population: Click "Create Population" to generate a new sampling frame with predation events (blue) and predation paths (orange)
  3. Draw Sample: Click "Draw Sample" to randomly sample cells. Red borders show sampled cells (origins, predation paths, and empty cells)
  4. Compare Approaches: Review unbiased estimates (counting only origins) vs. biased estimates (including predation paths)
  5. View Simulation: The simulation automatically runs after creating a population, showing distributions from repeated samples

Each cell represents a 1-hectare plot observed for 30 minutes. Pinnipeds surface with fish (probability p), then feeding continues across space and/or time until consumed. The model assumes Bernoulli(p) values (0 or 1) for simplicity, though actual counts can be any integer. This demonstration also assumes perfect detection; in reality, detection probability often varies with event characteristics.
