Pinniped Predation Sampling Demo

Demonstrating event duration bias in spatio-temporal sampling of predation events

How to Use This Demo

  1. Set Parameters: Choose temporal units (rows), predation probability (p), and sampling fraction
  2. Create Population: Click "Create Population" to generate a new sampling frame with predation events (dark blue) and foraging paths (light blue)
  3. Draw Sample: Click "Draw Sample" to randomly sample cells. Red borders show sampled cells (origins, foraging paths, and empty cells)
  4. Compare Approaches: Review unbiased estimates (counting only origins) vs. biased estimates (including foraging paths)
  5. View Simulation: The simulation automatically runs after drawing a sample, showing distributions from 1000+ repeated samples

About Predation Events & Event Duration Bias

Scenario: Each cell represents a 1-hectare water plot observed during a 30-minute period. Pinnipeds surface with newly caught fish with probability p. Once surfaced, feeding continues across space (horizontal), time (vertical down), or both (diagonal) until the fish is consumed.

Event Duration Bias: Longer events spanning multiple cells have higher detection probability than single-cell events. Unbiased estimation requires counting only event origins (dark blue cells showing "1"), not foraging paths (light blue, dashed border). Including foraging paths creates positive bias.

Note: This demonstration assumes perfect detection (probability of detection = 1, given a predation event occurs). In reality, detection probability is often less than 1 and may vary with event characteristics.

Statistical Formulas

For simple random sampling without replacement:

Estimated Total:

τ̂ = N × ȳ

where N = population size, ȳ = sample mean

Standard Error of the Total:

SE(τ̂) = N × √[(N - n)/(N - 1) × s²/n]

where s² = sample variance, n = sample size, (N-n)/(N-1) = finite population correction

95% Confidence Interval:

CI₉₅ = τ̂ ± 1.96 × SE(τ̂)

where 1.96 is the critical value from the standard normal distribution for 95% confidence

Sample Variance (for Bernoulli data):

s² = ȳ(1 - ȳ)

This simplified form applies when the response is binary (0 or 1)
